# Análisis de Ventas - TUP UTN

## Integrantes
- Carbone, Valentín

## Escenario
Escenario B – Análisis de Ventas de una Pequeña Empresa

## Dataset utilizado
Dataset simulado de ventas comerciales con información de producto, cantidad vendida, precio y fecha de venta.
Almacenado en /datos/ventas.csv.

## Descripción del proyecto
Script de análisis de ventas que calcula indicadores clave: ventas totales, producto más vendido y ventas por mes.
Genera un gráfico de evolución mensual de ventas guardado en /resultados/grafico_ventas.png.

## Instrucciones para ejecutar
1. Abrir Google Colab
2. Clonar el repositorio: git clone https://github.com/Valentin-Gif1/Analisis-Ventas-TUP.git
3. Navegar a la carpeta scripts: cd Analisis-Ventas-TUP/scripts
4. Ejecutar el script: python analisis_ventas.py
